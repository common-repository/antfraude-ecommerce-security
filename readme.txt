== Instalação ==

1 - Tenha o Woocommerce instalado e ativado.
2 - Acesse o painel administrativo do seu WordPressLog vá até Plugins -> Adicionar novo.
3 - Digite no campo de Busca "AntFraude".
4 - Encontre o AntFraude.
5 - Clique em instalar agora, após a instalação ative o plugin.
6 - Com seu cadastro ativo na plataforma antfraude.com gere seu token de acesso e adicione ao campo de token no plugin.

== Installation ==

1 - Have Woocommerce installed and activated.
2 - Access your WordPressLog administrative panel go to Plugins -> Add new.
3 - Enter "Plugin Anti Fraud" in the search field.
4 - Find the Anti Fraud Plugin.
5 - Click on install now, after the installation activates the plugin.
